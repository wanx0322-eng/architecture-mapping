# Pinterest 采集规则

1. 仅采集用户已登录页面中可见的公开 Pin 卡片。
2. 不读取 Cookie、密码、Local Storage 或账户隐私数据。
3. 不绕过 CAPTCHA、访问限制、登录墙或速率限制。
4. 每轮滚动后等待页面自然加载；连续五轮无新增即停止。
5. 保存 Pin URL、可见标题/描述、原站链接、缩略图 URL 和采集时间。
6. 下载的图片最长边缩放到 768px；不保留高清原图。
7. 记录 SHA-256 与 64 位 pHash；近重复保留清晰版本并合并来源。
8. 无关的思维导图、软件架构图、商业地图标记为 `rejected`。
9. 保留失败与暂停状态，允许后续断点续跑。
10. 数据集用于研究、分类和提示词抽象；不得暗示获得原作品版权。
