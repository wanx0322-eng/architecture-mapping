# 竞赛图纸四层分类

分类使用四个互相独立但有关联的层级：

1. `primary_category`：九类信息目的，只能选一个。
2. `drawing_subtype`：主类中的分析任务，可选一个主要子型。
3. `drawing_forms`：实际使用的图纸形态，可以多选。
4. `technical_execution`：投影、比例、线宽、图层、数据和输出参数。

不要用“效果图”代替图纸形态，也不要用“爆炸轴测”代替分析目的。

## 1. 场地区位

- `LOC-01 多尺度定位链`：国家—城市—片区—场地逐级定位。
- `LOC-02 城市形态与图底`：实体、街巷、开放空间、地块颗粒度。
- `LOC-03 交通与可达性`：轨道、公交、车行、慢行、时间圈。
- `LOC-04 土地利用与功能`：用地、公共服务、业态、开发强度。
- `LOC-05 蓝绿生态网络`：水系、绿地、廊道、斑块和生态连接。
- `LOC-06 地形水文气候`：等高线、坡度、汇水、风、日照、热环境。
- `LOC-07 视线地标界面`：视域、天际线、地标、门户与边界。
- `LOC-08 密度设施与数据`：人口、建筑密度、POI、热区和服务半径。

## 2. 历史文化

- `HIS-01 历时地图演变`：同尺度年代地图或阶段图。
- `HIS-02 遗产资源清单`：建筑、构筑物、景观要素和保护等级。
- `HIS-03 文化路径与节点`：仪式、游线、事件、人物和场所串联。
- `HIS-04 类型与形态演变`：街区、院落、建筑或景观原型变化。
- `HIS-05 记忆叙事拼贴`：档案、口述、照片、地图和时间轴。

## 3. 现状问题

- `ISS-01 实景照片诊断`：照片定位、编号、描边和问题卡片。
- `ISS-02 约束与机会`：SWOT、边界、法规、建设条件和潜力。
- `ISS-03 环境性能`：日照、风、热、噪声、雨洪、生态和污染。
- `ISS-04 基础设施与资源`：交通、市政、能源、水、废弃物和服务设施。
- `ISS-05 冲突风险与韧性`：洪水、灾害、安全、拥堵、断裂和脆弱性。
- `ISS-06 界面与剖面问题`：高差、边缘、首层、断面和空间尺度。
- `ISS-07 综合指标诊断`：多指标小图、评分、矩阵和综合结论。

## 4. 手绘构思

- `SKT-01 概念草图`：核心概念、力线、关键词和关系。
- `SKT-02 叙事拼贴`：地图、照片、人物、材质和文本碎片。
- `SKT-03 过程轨迹`：观察、推演、试错和设计决策记录。
- `SKT-04 隐喻图解`：生态、社会或空间概念的抽象转译。
- `SKT-05 手绘总平/剖面`：保留手绘线性与水彩层次的技术图。

## 5. 体块生成

- `MAS-01 加法与减法`：基础体块、挖空、切除和附加。
- `MAS-02 院落与缝隙`：开洞、内院、通廊和孔隙。
- `MAS-03 抬升架空退台`：顺应首层、景观或高度控制。
- `MAS-04 错动扭转连接`：体块位移、旋转、桥接和咬合。
- `MAS-05 地形适应`：嵌入、覆土、顺坡和台地化。
- `MAS-06 功能驱动塑形`：由功能邻接、流线或环境性能生成。
- `MAS-07 模块聚合`：单元复制、组合、变异和群体形成。

## 6. 功能流线

- `CIR-01 功能分区与邻接`：功能色块、泡泡图和邻接矩阵。
- `CIR-02 人车慢行组织`：行人、机动车、自行车和换乘。
- `CIR-03 出入口与可达性`：入口、服务半径、无障碍和消防。
- `CIR-04 垂直交通`：楼梯、电梯、坡道和楼层连接。
- `CIR-05 空间序列`：到达、进入、转换、停留和离开。
- `CIR-06 视线与导视`：视线、地标、可见性和路径选择。

## 7. 爆炸图

- `EXP-01 地图专题图层`：卫星、建筑、道路、绿地、水文、风险等。
- `EXP-02 景观系统分层`：地形、水、种植、铺装、设施和活动。
- `EXP-03 楼层与功能分解`：楼板、房间、交通和公共空间。
- `EXP-04 结构表皮机电`：结构、围护、节点、设备和管线。
- `EXP-05 建造与阶段`：施工、更新、季节或运营阶段。

## 8. 人群行为

- `BEH-01 用户画像`：年龄、角色、能力和需求。
- `BEH-02 活动密度与时间`：热区、昼夜、工作日/周末和季节。
- `BEH-03 移动与停留`：轨迹、停留点、速度和冲突。
- `BEH-04 场景脚本`：人物序列、行为面板和使用故事。
- `BEH-05 参与式与社会网络`：访谈、共创、关系和反馈。
- `BEH-06 无障碍与包容性`：不同身体能力和照护关系。

## 9. 效果图

- `VIS-01 写实空间渲染`：材质、光照、植被和真实生活场景。
- `VIS-02 竞赛拼贴`：模型、照片、人物和纸张纹理混合。
- `VIS-03 水彩/手绘氛围`：柔边、透明叠色和自然笔触。
- `VIS-04 轴测插画`：无灭点或弱透视的整体空间叙事。
- `VIS-05 剖透视/剖面场景`：结构、空间与活动同步可见。
- `VIS-06 夜景与事件氛围`：照明、节庆、运营和时间变化。
- `VIS-07 连续叙事帧`：同一空间的过程、季节或使用变化。

## 图纸形态多标签

可从下列形态中多选，并记录它们在版面中的角色：

- 区位地图、卫星叠加、GIS专题图、图底关系图、总平面；
- 轴测、爆炸轴测、透视、鸟瞰、剖面、剖透视、立面；
- 时间轴、流程图、网络图、桑基/关系流、矩阵、小多图；
- 实景照片分析、拼贴、行为分镜、数据图表、图例系统；
- 主图、辅助图、证据图、结论图、策略图、氛围图。
