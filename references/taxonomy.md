# 九类竞赛分析图分类体系

九类只表示图纸的主要信息目的。完成主类判断后，必须继续读取
`drawing-types.md`，确定一个 `drawing_subtype`、多个 `drawing_forms`，并按
`technical-detail-spec.md` 记录技术实现。

## 主分类判定

| 主类 | 判定重点 | 常见底图 | 常见表达 |
|---|---|---|---|
| 场地区位 | 城市关系、道路、肌理、生态、节点与轴线 | 卫星图/GIS | 分层地图、区位组图、图底关系 |
| 历史文化 | 时间演变、文脉、人物与事件 | 历史照片/地图 | 时间轴、拼贴、年代色阶 |
| 现状问题 | 对真实场地进行问题诊断 | 实景/航拍 | 描边、编号、卡片、剖面模块 |
| 手绘构思 | 以草图表达概念和关系 | 手绘/空白 | 放射、网状、图标、松弛线稿 |
| 体块生成 | 形体切割、错动、堆叠、架空与整合 | SU/Rhino | 同视角连续步骤、箭头、重点色 |
| 功能流线 | 功能分区、人车流、空间连接 | 模型/总平 | 色块、双层箭头、标签与节点 |
| 爆炸图 | 结构、表皮、楼板、景观或地图图层拆解 | 模型/GIS | 爆炸轴测、垂直引线、层级标注 |
| 人群行为 | 人群活动、停留、交往与微观动线 | 平面/轴测 | 人物剪影、行为面板、路径 |
| 效果图 | 方案氛围、拼贴、材质和生活场景 | 模型截图 | 拼贴、插画、水彩、叙事渲染 |

## 横向标签

- 底图：卫星地图、GIS、SU/Rhino、实景照片、航拍、手绘、空白。
- 投影：正交平面、等轴测30/30、军事斜投影45/45、30/60、透视、剖面。
- 布局：时间轴、放射、网状、模块化、2x3序列、爆炸分层、主图+侧栏。
- 风格：黑白灰+单色、马卡龙、莫兰迪、SASAKI、BIG、复古拼贴、暗黑汇报。
- 尺度：城市、片区、街区、场地、建筑、室内、构件。

## 四层输出

每次新分析至少输出：

1. 一个九类主类；
2. 一个带代码的子型，例如 `LOC-02 城市形态与图底`；
3. 一个或多个图纸形态及其角色，例如“图底关系图/主图、剖面/证据图”；
4. 图像细节、图底关系、艺术风格和技术执行字段。

当一张展板包含多个独立分析任务时，以主视觉结论确定主类，其他任务进入次级标签；
不要把整张展板粗略归为“效果图”。

主类必须唯一；次级标签可以跨类。分类证据不足时降低置信度并进入人工复核。
